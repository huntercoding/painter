Painter On Github



The painter plugin is a jQuery plugin that allows you to allows you to draw images basic images as of version1, and soon able to save images and have a full image editor suite online. The plugin is completely open-source, especially with module support to be released in version3. The plugin uses a mixture of jQuery and SVG support to work instead of being flash based. The tools the painter has to offer can be enabled or disabled to your liking and styling can be changed easily.

Changes

Version 1.0

  • Set up UI.
  • Added Pencil, Eraser, Circle, Box, and Line.
  • Added Color picker and color tools.

Version 1.1

  • Updated response time on pencil and eraser.
  • Added the finetune options
    • X and Y values to fix the offset.
    • Use as {finetune:{x:n,y:n}}
  • Added a killswitch to stubborn circles, rectangles, and lines.
  • Added the help menu to the menubar.
  • Minimized the amount of lines you have to type by automating it.
    • Automatically writes in canvas.js, color.js, and the stylesheet for you when the plugin loads.
